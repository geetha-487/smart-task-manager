# 🚀 Smart Task Manager

A modern productivity web application built using React, Node.js, and Express.

Users can:
- Register and Login
- Add Tasks
- Drag & Drop Tasks
- Track Task Progress
- View Analytics Dashboard
- Manage Priorities and Due Dates

---

# 📌 Features

✅ User Authentication  
✅ Task Management  
✅ Drag and Drop Kanban Board  
✅ Task Status Tracking  
✅ Priority Levels  
✅ Due Date Support  
✅ Analytics Dashboard  
✅ Responsive Premium UI
